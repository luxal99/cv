Projects

Real Estate Agency Business Application

A web application for real estate agencies. I was working on the frontend part of the application. Application has two sides, the business and the client-side. On the business side, we have two types of users admin and agent. Both of them can manage real estate. One of the interesting things about this project is that that real estate post can have up to 80 properties. And the form for creating a real estate post is complex and contains approximately 80 input fields(including select, text area, etc.). I used on the admin side features like, dynamic component, lazy loading, etc. On the client side, one of the greatest feature is our query builder for dynamic query generation.

Hotel Management System

This is a project for a client for the hotel management system. I worked on the Front-end part of the application. The project is pretty complex and has lots of business logic. In application, we can add a schedule of employees' working days and schedule of appointments in the custom calendar component which is one of the hardest things to develop on this application. Also, when I finished this project I have created a library from one component of the project, which is generic- material-table. Some of the interesting features of the project is: counting the earnings of the employee, expenses overview, appointment schedule overview, staff working days schedule overview, etc. Also, we have three types of users and in that case, we have features like protected components based on role.
